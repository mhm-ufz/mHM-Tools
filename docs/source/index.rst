=========
mHM Tools
=========

Tools to pre- and post-process data for and from mHM.

Quickstart
----------

Installation:

.. code-block:: bash

    pip install git+https://git.ufz.de/mhm/mhm-tools.git

For available components, see the :doc:`api/index`.

Usage
-----

See the `example scripts <https://git.ufz.de/mhm/mhm-tools/-/tree/main/examples>`_
in the GitLab repository for usage examples.
